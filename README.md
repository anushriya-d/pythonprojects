Welcome to **Python Projects** — a collection of simple and beginner-friendly Python programs that help you practice core concepts and have fun learning! 🚀

This repo includes a variety of small Python scripts that demonstrate basic programming logic, user input handling, randomness, loops, and more. All projects are standalone and easy to understand — perfect if you’re new to Python! 🐍
